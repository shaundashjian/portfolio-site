blackjack

A Java program to play a Blackjack game

Development Process

  1. First, I modeled the problem and drew out the key interfaces, classes, inheritance hierarchy, and state and behavior for each.
  2. Next, I created a project skeleton in Eclipse.
  3. Next, I used Test-Driven Development with JUnit to develop functionality, piece by piece. All unit tests are available in the GitHub repository as well.

Technology Used

I used Java SE 8 to develop this game, with JUnit 4 library for TDD.
